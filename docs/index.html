About the Game

Charlie and Charlinda's Wacky Adventure is a 2D top down roguelike game. In the game, you will play as one of 3 character classes as you fight your way through various rooms, each with enemies to kill and items to collect. Each character class has its own special ability, which you can leverage to gain an advantage against the enemies. The game provides player's with fast-paced, dungeon navigating gameplay!

Playing the Game

Step 1

Start the game and begin a new game run from the Main Menu.

Step 2

Choose the character class you want to play with. There are 3 characters, each with their own unique abilities!

Step 3

Fend off a ferocious pack of enemies as you navigate the various rooms. Defeat the final boss at the end to progress to the next floor!

Step 4

Earn achievements as you complete various challenges throughout the game.
